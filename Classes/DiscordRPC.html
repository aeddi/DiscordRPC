DiscordRPC

public class DiscordRPC

An object that manages an RPC connection to Discord through UDS and allows to interact with the Discord API using Swift types and methods.

  • connect()

    Init connection to the Discord RPC API, this will trigger: - on success: an EventReady event on the connectHandler - on failure: an EventClose event on the disconnectHandler

    Throws

    • RPCError.appSandboxed: If the current application is sandboxed
    • RPCError.socketCreation: If the creation of the socket failed
    • RPCError.udsNotFound: If unable to find the Discord UDS file
